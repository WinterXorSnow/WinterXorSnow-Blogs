#DP #题解 

[题目链接](https://www.luogu.com.cn/problem/P5999)

连续段DP板子题。

考虑转化问题成 $1$ 到 $n$ 的排列，满足 $p_1 = s$，$p_n = t$，要求 $p_i$ 要么同时大于两侧，要么同时小于两侧。然后常规的设计DP状态，令 $f[i][j]$ 表示前 $i$ 个数，形成 $j$ 段的方案数。答案显然在 $f[n][1]$。

若 $i = s/t$，则：

$f[i][j]=f[i-1][j]+f[i-1][j-1]$，什么意思呢？表示 $s/t$ 可以选择新开一段在开头/结尾，也可以和后面的位置相接。

若 $i \ne s/t$，则考虑三个操作：

1.新开一段

$f[i][j]=f[i-1][j-1]\times(j-[i>s]-[i>t])$，因为现在有 $j-1+1$ 个空，但是考虑到 $s,t$，因此要相应的减少

2.连在另一端前面/后面

仔细想下，肯定不合法。

3.连接两段

$f[i][j]=f[i-1][j+1]\times j$，因为现在有 $j$ 个空可以插，且无限制。

## Code
```cpp
#include<bits/stdc++.h>
using namespace std;
#define IOS ios::sync_with_stdio(false);cin.tie(0),cout.tie(0)
#define File(s) freopen(s".in","r",stdin);freopen(s".out","w",stdout)
#define LL long long
#define fi first
#define se second
const int N = 2e3 + 10;
LL f[N][N];
const LL mod = 1e9 + 7;
int n,s,t;
int main()
{
	IOS;
	cin >> n >> s >> t;
	f[1][1] = 1;
	for(int i=2;i<=n;i++){
		for(int j=1;j<=n;j++){
			if((i == s) || (i == t))
				f[i][j] = (f[i-1][j] + f[i-1][j-1]) % mod;
			else{
				f[i][j] = ((f[i-1][j+1] * j) % mod) + (f[i-1][j-1] * (j - (i>s) -(i>t))) % mod;
				f[i][j] %= mod;
			}
		}
	}
	cout << f[n][1];
	return 0;
}
```
