#题解 
# P2407 [SDOI2009] 地图复原 Sol
[题目链接](https://www.luogu.com.cn/problem/P2407)

思维题。观察到这道题没有SPJ且保证有解，因此方案数唯一。一开始考虑直接模拟，但是发现有点问题，因为无法定是顺时针转还是逆时针转。尝试人为的规定。

因为有解，所以同一行必定存在偶数个 `T` 并且两两配对（不然无解或不符合题意），列也是同理，然后就可以规定方向了。做完了。

## Code
挺有意思的实现

```c++
#include<bits/stdc++.h>
using namespace std;
#define IOS ios::sync_with_stdio(false);cin.tie(0),cout.tie(0)
#define File(s) freopen(s".in","r",stdin);freopen(s".out","w",stdout)
#define LL long long
#define fi first
#define se second
const int N = 805;
char c[N][N];
int n,m;
int suf[N][N],dwn[N][N];
int main()
{
	IOS;
	cin >> n >> m;
	for(int i=1;i<=n;i++)
		for(int j=1;j<=m;j++) cin >> c[i][j];
	for(int i=1;i<=n;i++){
		bool t = 0;
		for(int j=1;j<=m;j++){
			if(c[i][j] == 'T') t ^= 1;
			suf[i][j] = int(t);
		}
	}
	for(int j=1;j<=m;j++){
		bool t = 0;
		for(int i=1;i<=n;i++){
			if(c[i][j] == 'T') t ^= 1;
			dwn[i][j] = t;
		}
	}
	for(int i=1;i<=n;i++){
		for(int j=1;j<=m;j++){
			cout << 'o';
			if(suf[i][j]) cout << '-';
			else cout << ' ';
		}
		cout << "\n";
		for(int j=1;j<=m;j++){
			if(dwn[i][j]) cout << '|';
			else cout << ' ';
			cout << ' ';
		}
		cout << "\n";
	}
	return 0;
}
```
