#题解 
# P7515 [省选联考 2021 A 卷] 矩阵游戏 题解
[题目链接](https://www.luogu.com.cn/problem/P7515)

## 简单版本的题
如果没有 $a_{i,j}$ 的限制，怎么做？显然可以让边上一圈为 `0`，然后慢慢搞就行了，时间复杂度 $O(n\log n)$。

但是因为 $0\le a_{i,j} \le 10^6$，因此这个做法不可取

## 改变矩阵a
观察一下，可以发现，对于 $a$ 的任意一行，如果这个行依次 $+x,-x,+x,-x \dots$，那么显然对 $b$ 没有影响。列也是一样。因此考虑通过这种方式对 $a$ 进行调整。

因此，定义 $r_i$ 表示第 $i$ 行增加的值，$c_j$ 表示第 $j$ 列增加的值，那么 $A = \Delta a$ 就为：

$$
\begin{pmatrix}
r_1+c_1 & -r_1+c_2 & r_1+c_3 &\cdots \\
r_2-c_1 & -r_2-c_2 & r_2-c_3 &\cdots \\
\vdots & \vdots & \vdots & \ddots \\
r_n+c_1 & -r_n+c_2 & r_n+c_3 &\cdots
\end{pmatrix}
$$

那么对于一个位置，就变成了 $0\le a_{i_j}+A_{i,j} \le 10^6$

转换一下，变成 $-a_{i,j} \le A_{i,j} \le 10^6-a_{i,j}$

但是这样子 $r+c$ 与 $r-c$，难以处理，考虑对于 $i$ 为偶数的 $r_i$ 取反，对于 $j$ 为奇数的 $c_j$ 取反。因为在上面的调整中，并没有规定 $x$ 的正负，且显然对于 $-x$ 也是成立的。那么矩阵变成这样：

$$
\begin{pmatrix}
r_1-c_1 & c_2-r_1 & r_1-c_3 &\cdots \\
c_1-r_2 & r_2-c_2 & c_3-r_2 &\cdots \\
r_3-c_1 & c_2-r_3 & r_3-c_3 &\cdots \\
\vdots & \vdots & \vdots & \ddots \\
\end{pmatrix}
$$

这样子就变成了 $x-y$ 的形式。可以跑差分约束。

## 差分约束
[【模板】差分约束](https://www.luogu.com.cn/problem/P5960)，然后这题就可以一样建图了，有 $n+m$ 个点， $2nm$ 个约束，时间复杂度 $O((n+m)nm)$，跑不满，可以跑出来。

## Code
```c++
#include<bits/stdc++.h>
using namespace std;
#define IOS ios::sync_with_stdio(false);cin.tie(0),cout.tie(0)
#define File(s) freopen(s".in","r",stdin);freopen(s".out","w",stdout)
#define LL long long
#define fi first
#define se second
const int N = 305,MAX = 1e6;
LL a[N][N],b[N][N];
LL dis[N<<1];
int n,m;
struct edge{
	int to,val;
};
vector<edge> G[N << 1];
int in[N << 1];
bool vis[N << 1];
bool spfa(){
	G[0].clear();
	for(int i=1;i<=n+m;i++)
		G[0].push_back({i,0});
	memset(dis,0x3f,sizeof dis);
	memset(in,0,sizeof in);
	memset(vis,0,sizeof vis);
	dis[0] = 0;
	queue<int> q;
	q.push(0);
	vis[0] = 1;
	while(!q.empty()){
		int u = q.front();
		q.pop();
		vis[u] = 0;
		for(auto [v,w] : G[u]){
			if(dis[v] > dis[u] + w){
				dis[v] = dis[u] + w;
				if(vis[v] == 0){
					vis[v] = 1;
					in[v] ++ ;
					if(in[v] > n+m) return 1;
					q.push(v);
				}
			}
		}
	}
	return 0;
}
void solve(){
	for(int i=1;i<=n+m;i++){
		G[i].clear();
	}
	cin >> n >> m;
	for(int i=1;i<n;i++){
		for(int j=1;j<m;j++)
			cin >> b[i][j];
	}
	for(int i=n-1;i>=1;i--)
		for(int j=m-1;j>=1;j--)
			a[i][j] = b[i][j] - a[i+1][j] - a[i][j+1] - a[i+1][j+1];
	for(int i=1;i<=n;i++){
		for(int j=1;j<=m;j++){
			if((i + j) & 1){
				G[n+j].push_back({i,a[i][j]});
				G[i].push_back({n+j,MAX-a[i][j]});
			}
			else{
				G[i].push_back({n+j,a[i][j]});
				G[n+j].push_back({i,MAX-a[i][j]});
			}
		}
	}
	if(spfa()) cout << "NO\n";
	else{
		cout << "YES\n";
		for(int i=1;i<=n;i++){
			for(int j=1;j<=m;j++){
				if((i + j) & 1) cout << a[i][j] + dis[n+j] - dis[i] << " ";
				else cout << a[i][j] + dis[i] - dis[n+j] << " ";
			}
			cout << "\n";
		}
	}
}
int main()
{
	IOS;
	int T;
	cin >> T;
	while(T -- ){
		solve();
	}
	return 0;
}
```
