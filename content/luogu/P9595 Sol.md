#题解 
# P9595 「Daily OI Round 1」Xor Sol
[题目链接](https://www.luogu.com.cn/problem/P9595)

简单题，题解就写简单点了。

看到xor，考虑字典树。每一次xor相当于交换左右儿子。但是考虑每次交换左右儿子很慢，可以一次处理完，然后观察到 `tre[k][x]` 就是第 `k` 层xor `x` 的答案，然后做完了。

## Code
```cpp
#include<bits/stdc++.h>
using namespace std;
#define IOS ios::sync_with_stdio(false);cin.tie(0),cout.tie(0)
#define File(s) freopen(s".in","r",stdin);freopen(s".out","w",stdout)
#define LL long long
#define fi first
#define se second
const int M = 19,N = (1 << M) + 10;
struct node{
	int l,r,ans,p;
	node(){
		l = r = ans = p = 0;
	}
}tre[2][N];
int K = (1 << M);
int n;
void calc(node &p,node pl,node pr){
	if(pl.p) p.l = pl.l + pr.l;
	else p.l = pl.l;
	if(pr.p) p.r = pl.r + pr.r;
	else p.r = pr.r;
	p.ans = max(max(pl.ans,pr.ans),pl.r + pr.l);
	p.p = pl.p & pr.p;
	return ;
}
int main()
{

	IOS;
	cin >> n;
	int Q;
	cin >> Q;
	for(int i=1;i<=n;i++){
		int x;
		cin >> x;
		tre[0][x].l = 1;
		tre[0][x].r = 1;
		tre[0][x].ans = 1;
		tre[0][x].p = 1;
	}
	for(int i=1,nw=1,pre=0;i<=M;i++,nw^=1,pre^=1){
		for(int j=0;j<K;j++){
			calc(tre[nw][j],tre[pre][j],tre[pre][j ^ (1<<(i-1))]);
		}
	}
	int nw = 0;
	while(Q -- ){
		int x;
		cin >> x;
		nw ^= x;
		cout << tre[1][nw].ans << "\n";
	}
	return 0;
}
```
